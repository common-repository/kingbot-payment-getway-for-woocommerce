== Installation ==
This section describes how to install the plugin and get it working.
e.g.
1. Upload `woocommerce_pay_kingbot_ir_getway_wp.zip` to the `/wp-content/plugins/` directory
2. Extract It .
3. Delete 'woocommerce_pay_kingbot_ir_getway_wp.zip' File .
4. Go to Your Wordpress Admin -> Plugins and Active The Plugin .
5. Now Plugin Setting Has Been Added To woocommerce Getway setting .
